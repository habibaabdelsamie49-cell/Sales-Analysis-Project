# Sales-Analysis-Project
Analysis,Visualize and make insights on sales Data Using Excel,Power Query and Power Pivot.

## Overview
In this project I have cleaned, transformed, and analyzed sales data to build an interactive dashboard that highlights key business metrics and performance trends.

## Tools
- Excel
- Power Query
- Power Pivot

## What I Learned
Working on this project helped me improve my skills in data cleaning, data transformation, and data modeling. I also learned how to use Power Query and Power Pivot to turn raw data into meaningful insights and build interactive dashboards for decision-making.
